scrolls Sample¶

This is a demo site for scrolls, generated as part of Sphinx Themes Gallery.

Important

This sample documentation was generated on Jun 14, 2021, and is rebuilt weekly.

Quickstart¶

  1. Install this theme:

    $ pip install sphinx

  2. Set the following in your existing Sphinx documentation’s conf.py file:

    html_theme = 'scrolls'

  3. Build your Sphinx documentation with the new theme! 1
